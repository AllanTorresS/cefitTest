1. Layouts

Layouts

As opções de layout a seguir podem ser usadas para diferentes propósitos. Escolha a que melhor se adequa ao seu uso. Você pode escolher entre um cabeçalho de largura total ou uma barra de menus de altura total.

Fixar a barra de menu, se você quiser começar com uma barra de menus expandida. Quando você escolhe uma barra de menus expandida, ela é automaticamente reduzida em telas menores.
